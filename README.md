# Selenium Automation Test Suite | C# | Selenium WebDriver | NUnit

This project is a Selenium-based automation suite built using **C#**, **Selenium WebDriver**, and **NUnit**.  
It automates multiple UI interactions on the **Automation Practice Form** hosted at:
